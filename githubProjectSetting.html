#如何用同一个github账号在两台电脑上同步开发

##我们有时需要外出开发，将项目提交到github，然后希望家里的能同步，继续开发，配置步骤如下

###一、 安装git Bash

###二、 安装nodejs 

- 下载并安装nodejs
- 使用node -v 检查nodejs是否安装成功
- npm -v 来检查npm是否安装成功

###三、 生成本地密钥ssh(此处不能获取http)，复制链接，添加到github和gitos，添加完成后就没有github什么事了
生成本地密钥代码 

1. git config --global user.name your-name  
2. git config --global user.email "admin@qq.com"  
3. ssh-keygen -t rsa -C "admin@qq.com"

###四、 在git Bash中安装hexo和hexo-cli

1. npm install -g hexo  
2. npm install -g hexo-cli

###五、 将远程仓库克隆到本地 

1. git clone git@github.com:shilili0/blogs.git

###六、 在clone的目录下执行npm install 

1. npm install

###七、然后就完成了，启动服务器，输入localhost:4000,就可以看到clone下的内容了

1. hexo deploy
2. hexo server
